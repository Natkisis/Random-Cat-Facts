# Random-Cat-Facts
A beautiful web application that displays random cat facts with adorable cat images. Perfect for cat lovers and those needing a daily dose of feline cuteness!

✨ Features
 Random Cat Facts - Get interesting facts about cats from Cat Facts API
 Cat Images - Display random cat pictures from The Cat API
 Beautiful UI - Modern gradient background with smooth animations
 Fast Loading - Parallel API calls for optimal performance
 Responsive Design - Works perfectly on all devices
 Loading Animation - Elegant spinner while content loads
 Smooth Transitions - Fade-in animations for better user experience

🛠️ Technologies Used
 HTML5 - Semantic markup
 CSS3 - Modern styling with gradients and animations
 JavaScript (ES6+) - Async/await, Fetch API, DOM manipulation
 APIs:
  Cat Facts API - For random cat facts
  The Cat API - For random cat images

📦 Installation
 Clone the repository:
  git clone https://github.com/your-username/random-cat-facts.git
 Navigate to the project directory:
  cd random-cat-facts
 Open index.html in your browser or use a local server.

🎯 How to Use
 Open the application in your web browser
 The app will automatically load a random cat fact and image
 Click the "Another cat!" button to get new content
 Enjoy learning fun facts about cats!

🎨 Features Breakdown
 Core Functionality
 - Fetches data from two different APIs simultaneously
 - Handles errors gracefully

 User Interface
